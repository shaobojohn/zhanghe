Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app/src/main/AndroidManifest.xml
* assets/ => app/src/main/assets/
* libs/RootTools.jar => app/libs/RootTools.jar
* libs/android-support-v13.jar => app/libs/android-support-v13.jar
* libs/android-support-v4.jar => app/libs/android-support-v4.jar
* libs/eventbus-2.2.1.jar => app/libs/eventbus-2.2.1.jar
* libs/universal-image-loader-1.9.3.jar => app/libs/universal-image-loader-1.9.3.jar
* res/ => app/src/main/res/
* src/ => app/src/main/java/
